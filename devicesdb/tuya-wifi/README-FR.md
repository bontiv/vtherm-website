# Appareil Tuya WIFI

Versatile Thermostat fonctionne parfaitement avec les appareils Tuya WIFI (climatiseurs - AC). Ces appareils peuvent être configurés soit via [l'intégration officielle Tuya](https://www.home-assistant.io/integrations/tuya/), soit via [l'intégration Tuya locale](https://github.com/xZetsubou/hass-localtuya).

## Utilisation du climatiseur Tuya
Nous recommandons d'activer la [fonction démarrage/arrêt automatique](/fr/docs/feature-auto-start-stop/) ainsi que [la régulation automatique](/fr/docs/self-regulation/).
