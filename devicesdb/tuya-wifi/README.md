# WIFI Tyua Device

Versatile Thermostat work well with Tuya WIFI devices (Air Conditioner - AC). These devices can be configured with ether [the official tuya integration](https://www.home-assistant.io/integrations/tuya/) or [local tuya integration](https://github.com/xZetsubou/hass-localtuya).

## Use of Tuya Air Conditioner
We recommand to enable the [auto start-stop feature](/en/docs/feature-auto-start-stop/) and [the self regulation](/en/docs/self-regulation/).
